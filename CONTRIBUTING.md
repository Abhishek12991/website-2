# Guidelines

Follow the [branch-based workflow][flow].

## English

Specs are written in American English. Use:

* *color* ~~*colour*~~
* *gray* ~~*grey*~~

## Markdown

Use the [Kramdown syntax][kram].

## Images

Images must be PNG.
